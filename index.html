LIBRARY MANAGEMENT SYSTEM

“ Library Management System ” with Java, MySQL Database and Swing using development tools Eclipse IDE as a part of T.Y.B.Sc.(I.T) Curriculum. It is useful for libraries in any schools or similar educational institution to manage and maintain data in database through computerized system. The main feature of This Library Management System Using Java is that all the books available in the library can be borrow by the students. and also the borrowed books by the students can return in the library. Additionally, the application effectively maintains the details of users/students to whom books have been issued; it also records the issued date and return date. Technologies used : Eclipse IDE, Frontend UI : Java Swing, Backend : Java, MySql.
